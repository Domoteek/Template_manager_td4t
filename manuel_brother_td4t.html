📐 Calage des Étiquettes

Procédure de calage (Changement de rouleau) :
  1. Allumer l'imprimante et attendre qu'elle soit prête
  2. Appuyer sur la touche ² (touche de fonction) sur le clavier USB branché à l'imprimante
  3. Sélectionner le format d'étiquette correspondant (1, 2 ou 3)
  4. Attendre la fin du calibrage automatique
  5. Éteindre puis Rallumer l'imprimante lorsque le message vous y invite

🎯 Guide d'Utilisation Rapide

📸 Pour les Templates Image

Étape Action
1 Entrer le code du template désiré (ex: BOU01, PRO50, FET01)
2 Indiquer la quantité d'étiquettes
3 Appuyer sur ENTER pour imprimer

📝 Pour les Templates Texte et/ou Code-Barre

Étape Action
1 Entrer le code du template désiré
2 Saisir le code EAN (12 chiffres à renseigner)
3 Remplir les champs requis (libellé, prix)
4 Indiquer la quantité d'étiquettes
5 Appuyer sur ENTER pour imprimer

⌨️ Touches de Fonction

Touche Fonction
ESC Retour au menu principal
ENTER Validation
² (touche) CALIBRAGE (Démarrage / Changement rouleau) / Choix format

⚠️ Gestion des Erreurs

Erreur de Code-Barre :
Si un code-barre avec plus ou moins de chiffres que prévu est saisi, le système :
  • N'imprimera PAS la quantité demandée
  • Imprimera uniquement un message d'erreur
  • Interrompra la demande d'impression

⚠️ Résolution de Problèmes

Problème Solution
Étiquettes décalées Appuyer sur FORMFEED
Image ne s'imprime pas Vérifier format BMP (1-bit, 300 DPI)
Code EAN refusé Vérifier 13 chiffres exactement
Template introuvable Exécuter AUTO.BAS à nouveau

⚙️ Spécifications Techniques

Format BMP : 46x46 pixels • 1-bit (N&B) • 300 DPI
Codes-barres : EAN-13 (13 chiffres obligatoire)
Imprimante : Brother TD-4T • Format : 46x46mm
